# Chimes Project (Customized)

This repository contains a beautifully interactive web application that simulates physics-based chimes swinging from traditional roofs across various countries. 

This project is a customized fork by [arpittak027](https://github.com/arpittak027). 

## 🌟 Custom Tweaks & Features

I have made several modern enhancements and structural additions to the original project to elevate the user experience:

1. **Complete Global Roofs**: Generated and integrated custom roof structures for all previously missing countries (such as Turkey, Spain, South Korea, and Australia). Every country now features a visually distinct roof for the chimes to hang from!
2. **Glassmorphism UI**: Completely overhauled the user interface—including the top navigation bar, the settings tweakpane, and custom controls—with a premium, modern **glassmorphism** aesthetic (frosted glass, translucent backgrounds, and sleek borders).
3. **Custom Volume Controls**: Replaced the default audio slider with a custom, elegantly designed volume indicator featuring explicit "Up" and "Down" controls.
4. **Layout Optimizations**: Refined the positioning of the settings panel and titles to ensure elements do not intersect or overlap, providing a flawless desktop and mobile experience.

## 🛠️ Technologies Used
- HTML5 / CSS3 (Vanilla, CSS Variables, Flexbox, Glassmorphism)
- Vanilla JavaScript
- Tweakpane (for the physics & settings controls)
- Matter.js (for 2D physics simulation)

## 🏃 How to Run Locally

You can run this project easily using any local development server. For example, if you have Python installed:

```bash
python -m http.server 8765
```
Then navigate to `http://127.0.0.1:8765/` in your browser.
